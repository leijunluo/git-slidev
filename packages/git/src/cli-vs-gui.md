# CLI vs GUI

* Git GLI: 命令行界面（终端）。
* Git GUI: 图形界面。提供交互式 Git 操作。

<div class="mt-5">

# CLI 优势

* 任何环境下 CLI 都是相同的。
* CLI 相对于 GUI 功能是完整性的。
* 遇到了问题，更容易获得帮助。
</div>

<div class="mt-5">

# GUI 优势

* 在涉及到分支和提交历史的可视化时，提供了令人愉悦的交互体验。
* 非常直观的查看文件的差异。
</div>
